How to use

How to use each tool on this site

HTML Downloader

Enter the URL in the text box in the middle, then press the Download button to download the HTML data for that site.

SB3 Downloader

Enter the URL of your Scratch project into the text box in the middle, then press the Download button to download the project's .sb3 file. (You cannot download projects that are not published.)

QR Code Maker

Enter a URL or image into the middle text box, then press the Generate button to download a QR code containing that data.
